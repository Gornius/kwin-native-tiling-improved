# Native tiling improved

KWin script that modifies behavior of native KDE Plasma tiling functionality
together be more ergonomic in daily use.

## Features

### Unmaximize to Tile

When tiled window is being maximized it will remember the tile it was in, once
it has been unmaximized it will be tiled back to original location rather than
being restored to the location and size before tile.

### Raise together

When windows are tiled together, raising one of them (for example by clicking on
them) other windows will also be raised.
